# ONAP Release Mapping Artifacts

This repository stores the output artifacts generated by the
[onap-release-mapping-tool](https://github.com/modeseven-lfit/onap-release-mapping-tool).

It is a **passive data store** — no workflows run here. All content
is pushed by the tool repository's `mapping-production.yaml` workflow.

## Directory Structure

```text
data/
  artifacts/
    YYYY-MM-DD/
      manifest.json       # Release manifest (primary artifact)
      manifest.yaml       # YAML format
      manifest.csv        # CSV format (repositories only)
      manifest.md         # Markdown report
      metadata.json       # Run provenance
      README.md           # Auto-generated run summary
    latest -> YYYY-MM-DD  # Symlink to most recent
```
